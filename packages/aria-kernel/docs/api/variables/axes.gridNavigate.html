    키 매핑 (data grid model, no wrap): ArrowLeft/Right : 같은 row 의 prev/next cell. 양 끝에서 정지. ArrowUp/Down : 인접 row 의 같은 column index cell. 양 끝에서 정지. Home / End : 현재 row 의 첫/끝 cell. Ctrl+Home / End : grid 의 첫 row 첫 cell / 끝 row 끝 cell.
